Modulos del Sistema

📦 Inventario

Control de ingredientes, productos terminados, proveedores y trazabilidad por lote. Alertas de stock bajo y productos por caducar.

COFEPRISTrazabilidadLotes

📋 Recetas y Produccion

Costeo automatico de recetas, ordenes de produccion, descuento de ingredientes, control de merma y rendimiento.

CosteoMermaPlanificacion

🛒 Punto de Venta

Ventas con desglose fiscal IVA 0% (pan basico) y 16% (preparados). Tickets, corte de caja, multiples metodos de pago.

IVA 0%/16%Ley del ConsumidorTickets

🧾 Facturacion CFDI 4.0

Generacion de XML conforme al SAT con nodos Emisor, Receptor, Conceptos e Impuestos. Cancelacion con motivos oficiales.

SATCFDI 4.0XML

👥 Clientes

Gestion con datos fiscales completos: RFC validado con digito verificador, regimen fiscal, domicilio fiscal, uso CFDI.

RFC ValidadoRegimen Fiscal

👷 Empleados y Nomina

LFT: vacaciones (reforma 2023), aguinaldo, prima vacacional, PTU, horas extra dobles/triples. ISR Art. 96, IMSS, SDI.

LFTIMSSISRPTU

🏥 COFEPRIS

Control de temperaturas NOM-251, bitacora de limpieza, inspecciones sanitarias, control de plagas, licencias sanitarias.

NOM-251TemperaturasInspecciones

🏷️ Etiquetado NOM-051

Sellos de advertencia Fase 3: exceso calorias, azucares, grasas saturadas, grasas trans, sodio. Leyendas precautorias.

NOM-0515 SellosFase 3

📊 Reportes e Impuestos

Dashboard ejecutivo, reporte de ventas, IVA mensual para declaracion, ISR provisional, productos mas vendidos.

SATIVA MensualISR
